# validusbot-lua

A collection of Lua scripts, libraries, and API wrappers designed for ValidusBot development in Tibia.

This repository provides a structured scripting environment for creating advanced automation, cavebot logic, event-driven systems, inventory management, creature handling, cooldown tracking, hotkeys, map interactions, and custom bot features.

## Features

* Comprehensive Lua API references for ValidusBot
* Cavebot and waypoint automation utilities
* Creature, inventory, and container management
* Cooldown and spell tracking helpers
* Event proxy and callback systems
* Hotkey registration and custom actions
* HUD and interface scripting tools
* Modular architecture for easy extension
* Ready-to-use examples for Tibia automation

## Purpose

The goal of this repository is to simplify ValidusBot script development by providing reusable modules, documented APIs, and practical examples that help developers build reliable and maintainable Tibia automation scripts.

Whether you're creating custom cavebots, advanced hunting systems, utility tools, or event-driven automation, this repository serves as a foundation for ValidusBot Lua scripting.
